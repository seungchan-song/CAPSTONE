세종테크 DocSearch Pro v3.0 사용자 가이드

1. 제품 개요
DocSearch Pro는 기업 내부 문서를 AI 기반으로 검색하고 요약해주는 솔루션입니다. 사용자가 자연어로 질문을 입력하면, 관련 문서를 찾아 핵심 내용을 요약하여 답변합니다. 대규모 언어 모델(LLM)과 벡터 데이터베이스를 결합한 RAG 아키텍처를 채택하고 있습니다.

2. 시스템 요구사항
- 운영체제: Windows 10 이상, macOS 12 이상, Ubuntu 20.04 이상
- 메모리: 최소 8GB RAM (권장 16GB)
- 저장공간: 최소 50GB 여유 공간
- 네트워크: 안정적인 인터넷 연결 (API 호출 시 필요)

3. 설치 및 초기 설정
3.1 설치 패키지를 다운로드하여 실행합니다.
3.2 관리자 계정으로 로그인한 후, 문서 저장소 경로를 지정합니다.
3.3 임베딩 모델을 선택합니다. 한국어 문서의 경우 한국어 특화 모델 사용을 권장합니다.
3.4 인덱싱을 실행하면 지정한 경로의 문서가 자동으로 벡터 데이터베이스에 등록됩니다.

4. 주요 기능
4.1 자연어 검색: "올해 3분기 매출 현황 알려줘"와 같이 자연어로 질문하면 관련 문서를 검색하고 요약합니다.
4.2 문서 비교: 두 개 이상의 문서를 선택하여 핵심 차이점을 자동으로 분석합니다.
4.3 출처 표시: 답변에 사용된 문서의 출처(파일명, 페이지)를 함께 표시합니다.
4.4 접근 권한 관리: 부서별, 역할별로 문서 접근 권한을 세밀하게 설정할 수 있습니다.

5. 보안 설정
5.1 문서 암호화: 저장된 문서와 벡터 인덱스는 AES-256으로 암호화됩니다.
5.2 접근 로그: 모든 문서 접근 및 검색 이력이 감사 로그에 기록됩니다.
5.3 API 키 관리: 외부 LLM API 키는 별도의 비밀 저장소에 보관하며 코드에 직접 포함하지 않습니다.

6. 문의 및 지원
기술 지원이 필요한 경우 세종테크 고객지원 포털(support.sejongtech.example.kr)을 통해 문의하시기 바랍니다. 평일 오전 9시부터 오후 6시까지 전화 지원이 가능합니다.
