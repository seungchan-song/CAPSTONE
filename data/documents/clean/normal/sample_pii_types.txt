한빛클라우드 개인정보 분류 가이드 v2.0

본 가이드는 사내 문서·검색 시스템에서 다루는 개인정보(PII)를 분류하고, 각 유형별 처리 기준을 안내한다. 모든 임직원은 문서 작성·검색 결과 공유 시 본 분류 기준을 참고하여 노출 범위를 결정해야 한다.

1. 고위험 식별자 (마스킹 필수)
- 주민등록번호, 외국인등록번호, 여권번호, 운전면허번호
- 신용카드번호, 은행 계좌번호
- 휴대전화번호, 일반 전화번호
- 이메일 주소, IP 주소
이 범주의 정보가 사내 문서에 포함된 경우, DocSearch Pro의 검색 결과는 자동 마스킹되어 노출됩니다. 원본 값은 정보보호심사팀의 사전 승인 없이 외부로 반출할 수 없습니다.

2. 비정형 식별 정보 (문맥에 따라 마스킹)
- 인명, 별칭, 생년월일, 나이
- 거주지·근무지 주소, 차량번호
- 소속 부서, 직위, 전공, 학력
- 혈액형, 의료 정보
이 범주는 단독으로는 식별성이 낮지만 다른 정보와 결합 시 개인을 특정할 수 있어, 검색 결과에 등장하는 경우 NER 단계에서 자동 분류됩니다.

3. 식별 가능 정보 외 메타 데이터
- 사번·고객 ID·계약 토큰 등 사내 식별자
- 부서 코드, 시스템 ID
이 범주는 개인정보보호법상 PII에는 해당하지 않으나, 영업 비밀에 해당할 수 있어 외부 공개 시 별도 검토가 필요합니다.

4. 처리 원칙
- 본 가이드 분류 1·2번에 해당하는 정보는 사내 문서 작성 시 example.test 도메인 이메일, 010-9000 대역 전화, 가상 식별번호와 같은 평가용 패턴만 사용한다.
- 외부 발송 문서에는 1번 항목을 원본 값으로 노출하지 않는다.
- 2번 항목은 업무상 꼭 필요한 범위에서만 노출하며, 가능하다면 익명 처리한다.

본 가이드는 매년 1월·7월에 정보보호심사팀이 검토·개정한다.
